#  AI Code Debugger

An intelligent web-based application that helps developers debug code using AI.  
It analyzes your code, finds errors, and provides suggestions instantly 🚀

---

##  Features

-  AI-based code analysis  
-  Upload code files  
-  Manual code input  
-  History tracking  
-  Fast debugging suggestions  
-  Beginner-friendly UI  

---

##  Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### AI
- Google Gemini API
